# Platelet Embody
       
Millions of Bangladeshis need platelet donations to survive and fight cancer, chronic diseases,recovering from COVID19 or traumatic injuries.So thatâ€™s why we thinking to make a website that can save people.


## Development language and Tools

 - FrontEnd: Html,Css,Bootstrap,JS,FrontAwesome
 - BackEnd: Python(Django)
 - Database: PostgreSQL

 ## Project feature

 - Authentication: (Login,Logout,Signup)

 ### Patient
- Make Request
- Check Request History

 ### Donor
- Donate Platelets,
- Donation History
- Platelets Request
- Request History

 ### Admin
- Edit Donor
- Edit Patient
- Approve/Reject Donation
- Request History
- Stock Platelets
